# Applied Data Science Capstone Project

## Project Overview

This repository contains all notebooks, Python files, and supporting materials completed as part of the Applied Data Science Capstone Project.

The objective of this project is to analyze SpaceX Falcon 9 launch data and build machine learning models to predict whether the first-stage landing will be successful. The project follows the complete data science lifecycle, including data collection, data wrangling, exploratory data analysis, interactive visual analytics, dashboard development, and predictive modeling.

## Project Tasks

### 1. Data Collection

* Collected launch data using the SpaceX API.
* Extracted launch information through web scraping.

### 2. Data Wrangling

* Cleaned and transformed datasets.
* Handled missing values and prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)

* Performed statistical analysis.
* Identified patterns and trends in launch data.
* Created visualizations to explore relationships among variables.

### 4. SQL Analysis

* Queried launch datasets using SQL.
* Extracted insights through structured database operations.

### 5. Interactive Visual Analytics

* Developed interactive maps using Folium.
* Explored launch site locations and mission outcomes.

### 6. Dashboard Development

* Built an interactive dashboard using Plotly Dash.
* Enabled dynamic filtering and visualization of launch data.

### 7. Predictive Analysis

* Applied machine learning classification models.
* Evaluated model performance and selected the best-performing approach.
